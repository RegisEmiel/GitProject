Перейти в папку с проектом
cd e:/gitproject


Инициализация локального репозитория
git init


Добавляем все файлы
git add --all


Коммитим изменения в локальный репозиторий
git commit -m "comment3"


Добавить удаленный репозиторий
git remote add origin git@github.com:RegisEmiel/GitProject.git


Посмотреть результат связи с удаленным репозиторием
git remote -v


Пушим изменения на GitHub
git push -u origin master


Создание нового SSH-ключа
ssh-keygen -t rsa -b 4096 -C "email"


Запуск ssh-agent в фоновом режиме
eval "$(ssh-agent -s)"


Добавление приватного SSH-ключа в ssh-agent
ssh-add ~/.ssh/id_rsa
